Tags

As tags HTML são os blocos de construção de uma página web. Elas são usadas para definir e estruturar o conteúdo de um documento HTML. Cada tag HTML é composta por um nome de tag entre colchetes angulares "< >", algumas tags devem ser fechadas usando "/" seguido do nome da tag entre colchetes angulares.

Principais Tags HTML:

  • "html": Define o documento como um documento HTML.
  • "head": Contém metadados sobre o documento, como o título e links para arquivos CSS.
  • "title": Define o título do documento que aparece na aba do navegador.
  • "body": Contém o conteúdo visível da página, como textos, imagens e links.

Tags autofechadas e atributos

Algumas tags não têm conteúdo e são auto-fechadas.

As tags HTML podem ter atributos que fornecem informações adicionais sobre o elemento. Os atributos são incluídos na tag de abertura e têm a forma de pares nome-valor.

Títulos e Parágrafos

Os títulos são definidos pelas tags "h1" a "h6", onde "h1" é o título mais importante e "h6" é o menos importante. Eles ajudam a criar uma hierarquia de informações na página.

Os parágrafos são definidos pela tag "p" e são usados para agrupar blocos de texto.

Formatações de texto

  • Para deixar o texto em negrito, use a tag "b" ou "strong".
  • Para deixar o texto em itálico, use a tag "i".
  • Para deixar o texto sublinhado, use a tag "u".
  • Para deixar o texto taxado, ou riscado, use a tag "s".

Comentários

Para comentar uma parte do código, usa-se "!--" e para fechar "--".

Links

Para adicionar um link, use a tag "a" seguido do atributo "href" contendo o link do website. A tag de link é uma Tag auto-fechada.

Imagens

Para adicionar uma imagem, use a tag "img" seguido do atributo "src" seguido do arquivo, ou link, da imagem.

Listas

Listas são separadas em duas formas: Listas Não Organizadas, utilizando a tag "ul"; e Listas Organizadas, utilizando a tag "ol".

Para adicionar um item a lista, usa-se a tag "li" entre as tags de aberuta e fechamento da lista

Tabelas

Uma tabela em HTML é criada usando a tag "table". Dentro dela, você usa "tr" para definir uma linha, "th" para definir um cabeçalho de coluna e "td" para definir uma célula de dados.
